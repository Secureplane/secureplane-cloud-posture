# Wiz Cloud Native Application Protection Platform (CNAPP)

## Overview

This repository contains the implementation of a comprehensive Cloud Native Application Protection Platform (CNAPP) using Wiz.io. The platform provides end-to-end security coverage including cloud security posture management (CSPM), cloud workload protection (CWPP), data security (DSPM), and infrastructure as code (IaC) security.
